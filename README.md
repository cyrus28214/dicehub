# DiceHub

![DiceHub](./docs/DiceHub.jpg)

## 项目简介

DiceHub是一个微信小程序，专注于桌游评分、规则速查和社区交流。

## 项目特点

- 简洁的UI设计，扁平化风格
- 游戏规则速查，支持关键词搜索和分类浏览
- 社区交流功能，用户可以发表评论、对桌游进行打分
- 个人信息管理，用户可以管理个人信息，并通过微信的接口方便地登录和设置头像姓名
